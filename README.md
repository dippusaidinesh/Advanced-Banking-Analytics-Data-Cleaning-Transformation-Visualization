# Advanced Banking Analytics: Data Cleaning, Transformation & Visualization

##  Project Overview
This project focuses on performing **end-to-end data analysis** on a banking dataset (`advanced_banking_dataset.csv`). It covers the complete data analysis pipeline — from **data cleaning and preprocessing** to **data manipulation and visualization** — to extract meaningful business insights.

The goal of this project is to simulate a real-world banking analytics scenario and demonstrate how raw data can be transformed into actionable insights.

---

##  Tech Stack
- Python 
- Pandas 
- NumPy

- Seaborn

The dataset contains banking-related information such as:
